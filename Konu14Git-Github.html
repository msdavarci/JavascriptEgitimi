Git ve Github Kullanımı

Git : versiyon kontrol sistemidir.

Github : git sistemini kullanarak projelerimizi yönetip depolayan uygulamadır.

Önemli Git Terimleri

Repository Kısaca repo, dosyalarımızın bulunduğu projeler
Master Projemizin kararlı sürümü, ana gövdesi
Branch Git, ağaç mantığı ile çalışır ve proje birden fazla dala ayrılıp bu şekilde geliştirilebilir Sonrasında bu dallar yine master da birleştirilip tamamlanır
Merge Branc'lerdeki değişikliklerşn master da birleştirilme işlemidir.
Push Bilgisayardaki kodlarımızı github daki repoya(projenin kendi deposuna) göndermemizi sağlar
Pull Githubdaki projemizin son halini bilgisayarımıza çekmemizi sağlar.
.gitignore Git in takip etmemesini istedğimiz dosyaları (lokalde denediğimiz resimler vb) push işleminde repoya gönderilmesini engellememizi sağlar.
